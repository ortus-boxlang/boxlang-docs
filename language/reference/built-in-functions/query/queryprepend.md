# QueryPrepend

Adds a query to the beginning of another query

## Method Signature

```
QueryPrepend(query1=[query], query2=[query])
```

### Arguments

| Argument   | Type    | Required   | Description   | Default   |
| ---------- | ------- | ---------- | ------------- | --------- |
| `query1`   | `query` | `true`     |               |           |
| `query2`   | `query` | `true`     |               |           |
| ---------- | ------  | ---------- | ------------- | --------- |

## Examples

```
QueryPrepend(query1=[query], query2=[query])
```
