# Max

Return larger of two numbers

## Method Signature

```
Max(number1=[numeric], number2=[numeric])
```

### Arguments

| Argument   | Type      | Required   | Description       | Default   |
| ---------- | --------- | ---------- | ----------------- | --------- |
| `number1`  | `numeric` | `true`     | The first number  |           |
| `number2`  | `numeric` | `true`     | The second number |           |
| ---------- | ------    | ---------- | -------------     | --------- |

## Examples

```
Max(number1=[numeric], number2=[numeric])
```
