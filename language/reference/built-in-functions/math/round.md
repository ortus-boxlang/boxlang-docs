# Round

Rounds a number to the closest integer.

## Method Signature

```
Round(number=[numeric])
```

### Arguments

| Argument   | Type      | Required   | Description               | Default   |
| ---------- | --------- | ---------- | ------------------------- | --------- |
| `number`   | `numeric` | `true`     | The number to be rounded. |           |
| ---------- | ------    | ---------- | -------------             | --------- |

## Examples

```
Round(number=[numeric])
```
