# IsClosure

Determine whether a given object is a closure

## Method Signature

```
IsClosure(object=[any])
```

### Arguments

| Argument   | Type   | Required   | Description                         | Default   |
| ---------- | ------ | ---------- | ----------------------------------- | --------- |
| `object`   | `any`  | `true`     | The value to test for closure-ness. |           |
| ---------- | ------ | ---------- | -------------                       | --------- |

## Examples

```
IsClosure(object=[any])
```
