# IsJSON

Evaluates whether a string is in valid JSON (JavaScript Object Notation) data interchange format.

## Method Signature

```
IsJSON(var=[any])
```

### Arguments

| Argument   | Type   | Required   | Description                | Default   |
| ---------- | ------ | ---------- | -------------------------- | --------- |
| `var`      | `any`  | `true`     | The value to test for JSON |           |
| ---------- | ------ | ---------- | -------------              | --------- |

## Examples

```
IsJSON(var=[any])
```
