# IsIPv6

Determine whether the given hostname supports IPv6.

## Method Signature

```
IsIPv6(hostname=[string])
```

### Arguments

| Argument   | Type     | Required   | Description   | Default   |
| ---------- | -------- | ---------- | ------------- | --------- |
| `hostname` | `string` | `false`    |               |           |
| ---------- | ------   | ---------- | ------------- | --------- |

## Examples

```
IsIPv6(hostname=[string])
```
