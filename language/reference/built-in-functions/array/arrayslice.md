# ArraySlice

Extracts a sub array from an existing array.

## Method Signature

```
ArraySlice(array=[array], start=[integer], length=[integer])
```

### Arguments

| Argument   | Type      | Required   | Description   | Default   |
| ---------- | --------- | ---------- | ------------- | --------- |
| `array`    | `array`   | `true`     |               |           |
| `start`    | `integer` | `true`     |               | 1         |
| `length`   | `integer` | `false`    |               | 0         |
| ---------- | ------    | ---------- | ------------- | --------- |

## Examples

```
ArraySlice(array=[array], start=[integer], length=[integer])
```
