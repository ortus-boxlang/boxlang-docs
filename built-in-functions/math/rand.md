# Rand

Return a random double between 0 and 1

## Method Signature

```
Rand(algorithm=[string])
```

### Arguments

| Argument    | Type     | Required   | Description                                         | Default   |
| ----------- | -------- | ---------- | --------------------------------------------------- | --------- |
| `algorithm` | `string` | `false`    | The algorithm to use to generate the random number. |           |
| ----------  | ------   | ---------- | -------------                                       | --------- |

## Examples

```
Rand(algorithm=[string])
```
