# Atn

Returns the arc tangent (inverse tangent) of a number

## Method Signature

```
Atn(number=[numeric])
```

### Arguments

| Argument   | Type      | Required   | Description                                | Default   |
| ---------- | --------- | ---------- | ------------------------------------------ | --------- |
| `number`   | `numeric` | `true`     | The number to calculate the arc tangent of |           |
| ---------- | ------    | ---------- | -------------                              | --------- |

## Examples

```
Atn(number=[numeric])
```
