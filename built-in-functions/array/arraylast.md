# ArrayLast

Return first item in array

## Method Signature

```
ArrayLast(array=[array])
```

### Arguments

| Argument   | Type    | Required   | Description               | Default   |
| ---------- | ------- | ---------- | ------------------------- | --------- |
| `array`    | `array` | `true`     | The array to get the last |           |
| ---------- | ------  | ---------- | -------------             | --------- |

## Examples

```
ArrayLast(array=[array])
```
