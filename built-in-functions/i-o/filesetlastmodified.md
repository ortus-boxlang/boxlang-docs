# FileSetLastModified

Sets the last modified time of a file

## Method Signature

```
FileSetLastModified(file=[any], date=[any])
```

### Arguments

| Argument   | Type   | Required   | Description                  | Default   |
| ---------- | ------ | ---------- | ---------------------------- | --------- |
| `file`     | `any`  | `true`     | A file path or object        |           |
| `date`     | `any`  | `true`     | A date time object or string |           |
| ---------- | ------ | ---------- | -------------                | --------- |

## Examples

```
FileSetLastModified(file=[any], date=[any])
```
