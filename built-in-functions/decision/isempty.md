# IsEmpty

Determine whether a given value is empty

## Method Signature

```
IsEmpty(value=[any])
```

### Arguments

| Argument   | Type   | Required   | Description                      | Default   |
| ---------- | ------ | ---------- | -------------------------------- | --------- |
| `value`    | `any`  | `true`     | The value to test for emptiness. |           |
| ---------- | ------ | ---------- | -------------                    | --------- |

## Examples

```
IsEmpty(value=[any])
```
