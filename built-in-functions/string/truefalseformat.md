# TrueFalseFormat

Return Yes/No based on whether the input is true/false

## Method Signature

```
TrueFalseFormat(value=[any])
```

### Arguments

| Argument   | Type   | Required   | Description                                         | Default   |
| ---------- | ------ | ---------- | --------------------------------------------------- | --------- |
| `value`    | `any`  | `true`     | The value to check for true/false and return Yes/No |           |
| ---------- | ------ | ---------- | -------------                                       | --------- |

## Examples

```
TrueFalseFormat(value=[any])
```
