# Function: `IsObject`

Determines whether a value is an object.

True, if the value represents a object. False if the value is any other type of data, such as an
 integer, string, date, or struct.

## Method Signature

```
IsObject(value=[any])
```

### Arguments

|----------|------|----------|-------------|---------|
| `value` | `any` | `true` | The value to test |  |
